# 青绿山水贡献动画：概念图提示词

将下面整段提示词复制到 Codex 桌面端的图像生成对话中使用。

```text
为深色 GitHub 个人主页创作一张横向、无边框的青绿山水概念图。这是贡献动画的视觉底稿，先不要做像素风。

画面是宁静的夜景中国山水：近黑色炭灰背景自然融入页面，没有纸张质感、卡片、画框或边界；远景为墨靛色群山，中景为石青、石绿和青瓷色的层叠山脊，低处是一条安静蜿蜒的河谷，山间有几缕淡月光照亮的薄雾。整体借鉴《千里江山图》的青绿矿物颜料气质，但呈现为精致、克制的数字概念绘画。

请把贡献格子的节奏巧妙融入地貌，而不是在画面上叠一层日历：细小的青玉光点沿山脊连成若隐若现的灵脉；少量光点在河谷中化为水面粼光；只有极少数最亮的点在云雾缝隙里成为低饱和的金白色星火。光点应像山水本身的生命纹理，能隐约暗示 GitHub 贡献日历的节奏，但绝不能形成明显的规则网格、方块阵列或覆盖层。

构图必须是强烈的横向全景，视线沿一条安静的青玉灵脉从左向右流动。中部河谷保持适度留白，为未来放入真实贡献格子预留空间；山、水、雾在前中后三层中清晰分离。氛围为安静的午夜修仙世界，发光克制、精致、有空间感，而非戏剧化奇幻场景。

配色仅使用炭黑、墨靛、石青石绿、青瓷玉色，并极少量使用旧金和月白。不要文字、汉字、标志、水印、UI、日历线、可见方格、人物、动物、建筑、飞剑、边框或卡片。避免霓虹、高饱和、写实摄影、泛滥的水彩模糊、通用奇幻壁纸感和扁平装饰纹样。
```

## 生成后检查

- 山水是完整画面，而非“背景 + 一层方格”。
- 玉色光点首先像山脊灵脉，其次才隐约暗示贡献节奏。
- 河谷中部有可用于贡献格子的留白，不能被山、雾或高亮遮满。
- 画面必须没有任何可读文字或边框。
- 这张图只用于确认构图和色彩；获确认后，再转译成真正的泰拉瑞亚式硬边像素风 SVG 动画。
